1. 🔐 Tela de Login

A primeira tela do sistema. Aqui o usuário acessa sua conta ou cria uma nova.

Como fazer login:

  1. Digite seu email e senha
  2. Clique em 🔑 Entrar
  3. O sistema carrega automaticamente os dados do seu mercado

Como criar uma conta:

  1. Clique em "Criar conta" (link abaixo do botão Entrar)
  2. Preencha email, senha (mínimo 6 caracteres) e nome do estabelecimento
  3. Clique em 📝 Criar Conta
  4. O acesso é imediato após o cadastro

Esqueci minha senha:

  1. Digite seu email no campo de login
  2. Clique em "Esqueci minha senha"
  3. Um email de recuperação será enviado para redefinir a senha

Botão Sair:

  • No topo do sistema, clique em 🚪 Sair para desconectar
  • Seus dados ficam salvos e protegidos para o próximo acesso

2. 🏪 Seletor de Mercado

Logo abaixo do cabeçalho, aparece o seletor de mercado.

  • Cada usuário vê apenas os mercados associados à sua conta
  • Se tiver acesso a mais de um mercado, pode alternar entre eles
  • A opção "Todos os mercados" mostra os produtos de todas as unidades
  • Ao selecionar um mercado, todo o sistema filtra automaticamente: dashboard, alertas, lista e exportação

3. 📊 Resumo do Estoque (Dashboard)

Painel com 4 indicadores em tempo real:

IndicadorCorSignificado
TotalCinzaQuantidade total de produtos cadastrados
VencidosVermelhoProdutos com validade expirada
CríticosAmareloProdutos que vencem nos próximos 25 dias
OKVerdeProdutos com validade acima de 25 dias

Os números atualizam automaticamente sempre que um produto é adicionado, editado ou excluído.

4. ➕ Adicionar Produto

Seção para cadastrar novos produtos no estoque.

Campos:

CampoDescriçãoObrigatório
Código de BarrasCódigo numérico do produto (digitado ou escaneado)✅ Sim
Nome do ProdutoNome descritivo (ex: "Coca-Cola 2,5L")✅ Sim
FornecedorNome do fornecedor (ex: "Coca-Cola")Não
LoteIdentificação do lote (ex: "A24", "B12")✅ Sim
QuantidadeQuantidade de unidadesNão
ValidadeData de validade do produto✅ Sim

Preenchimento automático:

  • Ao digitar ou escanear um código de barras já cadastrado no catálogo, o sistema preenche automaticamente o nome e o fornecedor
  • O campo fica com borda verde por 2 segundos quando encontra o produto
  • Campos lote, quantidade e validade sempre devem ser preenchidos manualmente

Como salvar:

  1. Preencha os campos
  2. Certifique-se de que um mercado está selecionado no seletor
  3. Clique em 💾 Salvar Produto
  4. O produto aparece imediatamente na lista e no dashboard

5. 📷 Escanear Código de Barras

O sistema possui scanner integrado que usa a câmera traseira do celular.

Como usar:

  1. Clique em 📷 Escanear
  2. Autorize o uso da câmera (aparece apenas na primeira vez)
  3. Aponte a câmera para o código de barras do produto
  4. O código é lido automaticamente e preenchido no campo
  5. Se o produto estiver no catálogo, nome e fornecedor preenchem sozinhos

Para parar o scanner:

  • Clique em ⛔ Parar para fechar a câmera
Requisitos: O sistema deve estar acessado via HTTPS (obrigatório para a câmera funcionar). Funciona em qualquer celular com câmera.

6. ✏️ Editar Produto

Seção para alterar dados de um produto já cadastrado.

Como editar:

  1. Na lista de produtos, clique no botão ✏️ Editar do produto desejado
  2. Os dados preenchem automaticamente nos campos de edição (código, lote, validade)
  3. Altere o que for necessário
  4. Clique em ✅ Atualizar
  5. Uma mensagem confirma "Produto atualizado com sucesso!"

Campos editáveis:

  • Código de barras
  • Lote
  • Data de validade

7. 🗑️ Excluir Produto

Cada produto na lista possui um botão 🗑️ Excluir.

Como excluir:

  1. Clique em 🗑️ Excluir no card do produto
  2. Uma mensagem de confirmação aparece: "Tem certeza que deseja excluir [nome]?"
  3. Clique em OK para confirmar ou Cancelar para manter
  4. O produto é removido permanentemente do estoque
⚠️ Atenção: A exclusão é definitiva. O produto não pode ser recuperado após confirmação.

8. ⚠️ Alertas

Painel com 3 categorias de alerta. Clicável — ao clicar, expande e mostra os detalhes dos produtos.

❌ Vencidos

  • Produtos com validade expirada
  • Ao clicar, mostra cada produto com "venceu há X dia(s)"
  • Ação recomendada: retirar do estoque imediatamente

⚠️ Críticos

  • Produtos que vencem nos próximos 25 dias
  • Ao clicar, mostra cada produto com "vence em X dia(s)" ou "vence hoje!"
  • Ação recomendada: priorizar a venda ou fazer promoção

✅ OK

  • Produtos com validade acima de 25 dias
  • Ao clicar, mostra cada produto com "X dia(s) restantes"
  • Estoque saudável, sem ação necessária

Informações exibidas em cada alerta:

  • Nome do produto
  • Fornecedor
  • Lote
  • Data de validade
  • Dias restantes ou dias vencidos

9. 📋 Lista de Estoque

Exibe todos os produtos cadastrados em formato de cards.

Informações de cada card:

  • Nome do produto
  • Fornecedor
  • Mercado
  • Código de barras
  • Lote
  • Quantidade
  • Data de validade
  • Botões Editar e Excluir

Cores dos cards:

Cor da bordaSignificado
🟢 VerdeProduto OK (validade acima de 25 dias)
🟡 AmareloProduto crítico (vence em até 25 dias)
🔴 VermelhoProduto vencido

Busca / Filtro:

  • No topo da lista há um campo de busca 🔍
  • Digite qualquer palavra para filtrar em tempo real
  • Busca por: nome, código de barras, lote ou fornecedor
  • A filtragem é instantânea conforme você digita

10. 📊 Exportar Relatório (CSV)

O sistema permite exportar os dados do estoque em formato CSV (abre no Excel, Google Sheets, etc).

Como exportar:

  1. Selecione o mercado desejado no seletor (ou "Todos")
  2. Clique no botão 📊 Exportar CSV (canto superior direito da lista)
  3. O arquivo é baixado automaticamente

Conteúdo do relatório:

ColunaDescrição
NomeNome do produto
FornecedorNome do fornecedor
MercadoMercado associado
Código de BarrasCódigo do produto
LoteIdentificação do lote
QuantidadeQuantidade em estoque
ValidadeData de validade (DD/MM/AAAA)
StatusOK, CRÍTICO ou VENCIDO

Nome do arquivo: estoque_[mercado]_[data].csv

Exemplo: estoque_Mercado Aliança_2026-06-14.csv

11. 🔒 Segurança e Privacidade

Isolamento de dados:

  • Cada usuário vê apenas seus próprios produtos
  • Nenhum cliente tem acesso aos dados de outro cliente
  • A separação é garantida no nível do banco de dados

Sessão:

  • O login é mantido mesmo se fechar o navegador
  • Para desconectar, use o botão 🚪 Sair

Senhas:

  • Mínimo de 6 caracteres
  • Armazenadas de forma criptografada (nunca são visíveis)
  • Recuperação por email disponível

12. 📱 Uso no Celular

O sistema é 100% responsivo e funciona em qualquer celular ou tablet.

Dicas para uso mobile:

  • Adicione o site à tela inicial do celular para acesso rápido
  • Use o scanner para agilizar o cadastro de produtos
  • A câmera traseira é usada automaticamente para leitura de códigos

Como adicionar à tela inicial:

Android (Chrome):

  1. Abra o sistema no Chrome
  2. Toque nos 3 pontinhos (⋮) no canto superior
  3. Selecione "Adicionar à tela inicial"

iPhone (Safari):

  1. Abra o sistema no Safari
  2. Toque no botão de compartilhar (📤)
  3. Selecione "Adicionar à Tela de Início"

13. ❓ Perguntas Frequentes

P: Posso usar em mais de um mercado com o mesmo login?
R: Sim, se o administrador associar mais de um mercado ao seu login, o seletor mostrará todas as opções.
P: Se eu excluir um produto por engano, consigo recuperar?
R: Não. A exclusão é definitiva. Sempre confira o nome do produto na mensagem de confirmação antes de excluir.
P: O scanner não funciona, o que fazer?
R: Verifique se o sistema está acessado por HTTPS e se você autorizou o uso da câmera no navegador.
P: Posso acessar de vários dispositivos ao mesmo tempo?
R: Sim, o sistema é acessível de qualquer dispositivo com navegador.
P: Como cadastro um produto novo no catálogo?
R: Entre em contato com o suporte para inclusão de novos produtos no catálogo de códigos de barras.
P: O relatório CSV abre no Excel?
R: Sim. Basta dar duplo clique no arquivo baixado ou abrir pelo Excel/Google Sheets.
